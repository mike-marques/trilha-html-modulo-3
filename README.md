# 📺 Site Estilo Wikipédia moderno sobre Séries de TV

## 📝 Descrição
Este é um projeto de estudo do curso **HTML da DIO.me**, com o objetivo de criar um site inspirado na **Wikipédia**, mas com **layout moderno**.  

O foco principal foi **HTML semântico**, garantindo uma estrutura clara e acessível. O CSS foi usado **somente de forma complementar**, não sendo o foco do projeto.

O site apresenta informações sobre algumas séries de TV populares:  
🎬 **Friends**  
🎬 **Game of Thrones**  
🎬 **Everybody Hates Chris**  
🎬 **How I Met Your Mother**

---

## 🎯 Objetivo
- Aprender e aplicar **HTML semântico** corretamente  
- Organizar o conteúdo usando:  
  - `<header>` e `<footer>`  
  - `<main>` e `<section>`  
  - `<aside>` para barras laterais e índice do artigo  
  - `<nav>` para menus de navegação  
  - `<figure>` e `<figcaption>` para imagens  
  - `<table>` para elenco de personagens  
  - `<h1>` a `<h3>` para títulos e subtítulos  
  - `<form>`
  
- Criar um site **claro, acessível e navegável**
